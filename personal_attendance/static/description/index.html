Personal Attendance

Overview of the personal work time of the logged in user.

This module will add a new menu entry in the Attendance menu, which is named "My Working Time". From this menu the logged in users will be able to see their work times. Other employees working time won't be displayed. If the user rights are set to "Manual Attendance" the entries will be readonly.

Supported Languages

  • English
  • German
  • Spanish
  • French
